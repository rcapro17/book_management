O que é o curso?

O curso de Desenvolvimento de Sistemas capacita os alunos a criarem softwares, aplicativos, sites e sistemas, preparando-os para o mercado tecnológico.

Objetivo do Curso

Formar profissionais aptos a desenvolver, testar, implantar e manter sistemas computacionais, tanto para web quanto para dispositivos móveis.
